  1. AI 종목 리포트 /report

  현재 /analysis의 AI 분석을 더 깊게 확장한 버전

  무료: 월 3개 종목 리포트
  유료(9,900원/월): 무제한 + PDF 다운로드 + 과거 리포트 열람
  - 재무제표 5가지 축 시각화 (가치/성장/배당/건전성/모멘텀)
  - 동종업계 비교 테이블
  - AI 요약: "이 종목, 지금 사도 될까?"
  - 한국에 이런 서비스가 없음 → 가장 빠르게 유료 전환 가능

  2. 종목 스크리너 /screener

  조건에 맞는 종목을 필터링해서 찾아주는 도구

  무료: PER, PBR, 시총 등 기본 5개 필터
  유료(9,900원/월): AI 스코어, 외국인 수급, 기관 매매,
                     재무 성장률, 배당률 등 20개+ 필터
  - "PER 10 이하 + 배당률 3% 이상 + AI 점수 80점 이상" 같은 조합
  - 결과를 관심목록에 바로 추가
  - 알파스퀘어도 스크리너가 유료 → 검증된 수익 모델

  3. 포트폴리오 분석 /portfolio

  내 보유 종목을 등록하면 AI가 종합 분석

  무료: 1개 포트폴리오 (종목 10개)
  유료(7,900원/월): 다중 포트폴리오 + 리밸런싱 제안 + 섹터 분포
  - 한투 API 연동으로 실제 보유 종목 자동 불러오기 (이미 인프라 있음)
  - "당신의 포트폴리오는 IT에 72% 집중되어 있습니다"
  - 리스크 점수, 예상 배당금, 섹터 분산도

  4. 투자 시뮬레이션 /simulation

  과거 데이터로 "만약 그때 샀다면?" 백테스트

  무료: 단일 종목 시뮬레이션
  유료(9,900원/월): 포트폴리오 백테스트 + AI 전략 추천
  - "삼성전자를 2024년 1월에 100만원 어치 샀다면 → 현재 XX만원"
  - 적립식 vs 일시 투자 비교
  - AI 추천 전략 시뮬레이션 결과

  5. 프리미엄 알림 /alerts

  목표가/급등/급락/뉴스 알림

  무료: 3개 종목 알림
  유료(4,900원/월): 무제한 + AI 이상 신호 감지
  - "삼성전자 72,000원 도달 시 알림"
  - "외국인 3일 연속 순매수 시 알림"
  - AI 이상 감지: 거래량 급증, 급등락 패턴

  6. 커뮤니티 /community

  종목별 토론 + 분석 공유

  무료: 읽기/쓰기
  유료(4,900원/월): 광고 제거 + 프리미엄 배지 + 인기 분석가 전용 콘텐츠
  - 네이버 종토방의 대체제 (신뢰도 있는 버전)
  - 분석 작성자의 실제 수익률 표시 (한투 API)
  - 장기적으로 광고 수익
